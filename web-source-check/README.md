# CARPLUS AUTOSALE - Paquete de Sitio Web

## Contenido del Paquete

Este paquete contiene:

1. **Tu sitio web original** - Todos los archivos de tu sitio actual (carpeta principal)
2. **prequalify.html** - Formulario de pre-calificación listo para usar

## Instrucciones de Instalación

### Paso 1: Sube los archivos a tu hosting

1. Accede al panel de control de tu hosting (Hostinger, cPanel, etc.)
2. Ve al **Administrador de Archivos** o usa **FTP**
3. Navega a la carpeta `public_html` (o la carpeta raíz de tu sitio)
4. Sube todos los archivos de este paquete

### Paso 2: Verifica que funcione

1. Visita tu sitio web: `https://carplusautosalesgroup.com`
2. Visita el formulario: `https://carplusautosalesgroup.com/prequalify.html`

### Paso 3: Agrega un enlace al formulario en tu sitio

Puedes agregar un botón o enlace que lleve a tus clientes al formulario de pre-calificación.

**Ejemplo de enlace:**
```html
<a href="/prequalify.html">Pre-Calificación</a>
```

**Ejemplo de botón:**
```html
<a href="/prequalify.html" style="background: #1e3a8a; color: white; padding: 15px 30px; border-radius: 8px; text-decoration: none; font-weight: bold;">
  Pre-Califícate Ahora
</a>
```

## Configuración del Formulario

El formulario `prequalify.html` ya está configurado para enviar datos a tu CRM en:
`https://crm.carplusautosalesgroup.com`

### Campos del Formulario:
- ✅ Nombre y Apellido
- ✅ Email y Teléfono
- ✅ Fecha de Nacimiento
- ✅ Número de ID/Licencia
- ✅ SSN (últimos 4 dígitos)
- ✅ Dirección completa
- ✅ **Tiempo en Dirección (Años y Meses)** - NUEVO
- ✅ Tipo de Vivienda (Dueño/Renta/Vivo con familiares)
- ✅ Monto de Renta (si aplica)
- ✅ Nombre del Empleador
- ✅ **Tiempo con Empleador (Años y Meses)** - NUEVO
- ✅ Tipo de Ingreso
- ✅ Ingreso Neto
- ✅ Frecuencia de Pago
- ✅ Enganche Estimado
- ✅ **Subida de múltiples documentos de ID** (combina en un PDF)

## Cuando un Cliente llena el Formulario:

1. Los datos aparecen automáticamente en tu CRM → Sección "Pre-Qualify"
2. Recibes un email de notificación
3. Puedes convertir el lead en cliente con un clic
4. Todos los datos se transfieren automáticamente al nuevo cliente
